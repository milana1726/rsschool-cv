  1. Train-A (Team Project)
    • Project Overview: Train-A is a platform developed by our team to manage train trips with three user roles: anonymous users (can search routes), authorized users (can book and view trips), and resource managers (can manage trips and schedules).
      Built with Angular 18 using standalone components, NgRx for state management, and Angular Material for UI, the application features CRUD operations for stations and carriages, route management, and ride scheduling.
      The platform also integrates Leaflet for maps and uses Role-Based Route Guards for security.
    • Technical Approach:The application was structured with standalone components, removing the need for NgModules, allowing for better modularization. We used NgRx with the facade pattern to simplify state management interactions, while NgRx Effects managed API calls in combination with RxJS. Testing relied on Angular CLI, Jasmine, and Karma for robust verification of components and services.
    • My Contributions - I was responsible for:
      • Project Setup: Setting up the Angular project with essential development tools, including Prettier and ESLint for code quality.
      • UI Integration: Adding Angular Material to the project to ensure a polished and consistent UI.
      • Admin Pages: Implementing an admin page for managing stations, including CRUD functionalities.
      • User Pages: Implementing the User Profile page and Orders page, enabling users to manage and view their booking details.
    • Challenges & Solutions:
      • Complex Routing: Implemented Role-Based Route Guards to control access based on user roles, securing different areas of the application.
      • Time Zone Handling: Leveraged JavaScript’s Date object for basic time management, with plans to incorporate Moment.js for enhanced control across time zones.
    • Key Skills Gained:
      • Effective Team Collaboration: Managed task division and tracking with Trello and communication on Discord to ensure smooth progress.
      • Angular Standalone Components: Gained experience with Angular's standalone components for a more flexible and modularized architecture.
      • Dynamic Routing Management: Built a secure routing structure to handle multiple user roles dynamically.
